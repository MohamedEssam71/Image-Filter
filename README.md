# Image-Filter
Image filters program one for grey images other for color one.

It uses small image with bitmap format and has the following features :

1. Black & White Filter
2. Invert Filter
3. Merge Filter
4. Flip Image
5. Rotate Image
6. Darken and Lighten Image
7. Detect Image Edges
8. Enlarge Image
9. Shrink Image
10. Mirror Image
11. Shuffle Image
12. Blur Image
